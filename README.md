*An end-to-end machine learning project engineered to identify high-risk credit card holders before they cancel services. This allows financial institutions to run targeted, high-ROI retention marketing campaigns rather than wasting budget.*  
* **The Tech:** Python, Scikit-Learn, XGBoost, Pandas, Matplotlib, Seaborn.  
* **Engineering Depth:** Performed comprehensive Exploratory Data Analysis (EDA), handled missing values and feature transformations, and built a tuned predictive model evaluated against key classification metrics (Precision, Recall, ROC-AUC) to optimize cost-benefit trade-offs.

#### 🔍 Natural Language Processing (NLP) Semantic Analysis Pipeline
*An AI application designed to read, clean, and extract underlying semantic intent, patterns, and categories from unstructured business text and communication logs.*  
* **The Tech:** Python, Hugging Face Transformers, Tokenization pipelines, Advanced Text Cleaning, Regular Expressions (Regex).  
* **Engineering Depth:** Implemented text preprocessing steps (handling special characters, stop-word adjustments) and leveraged pre-trained transformer architectures to extract actionable business patterns from qualitative text datasets.

#### 🐾 Pawsome Care & Tracking App (Stanford Code in Place — Final Selection)
*A daily routine tracking and scheduling software ecosystem built to move manual management into an organized digital application. Developed originally as a Python CLI backend and mapped out into a beautiful interactive frontend.*  
* **The Tech:** Python, CSV File I/O, Lists/Dictionaries, Object-Oriented Architecture, HTML5, CSS3 (Sage/Amber variables), JavaScript.  
* **Engineering Depth:** Implemented robust local data persistence via custom CSV reading/writing routines, mapped structured relational data objects (routines, logs, reminders), and engineered responsive UI views for tracking real-time daily metrics.
