# PawRoutine — Pet daily care tracker

Small project built as part of Stanford Code in Place. This repo contains:
- main.py — Python CLI app for tracking pet tasks & logging (file-based CSV)
- pet_care_app_final.html — a static, interactive frontend demo (self-contained HTML/CSS/JS)

Why this project
- Demonstrates product-minded engineering: UI (HTML demo), CLI app, CSV persistence, and small UX considerations.
- Good example to showcase full-stack prototyping and small product demos for consulting clients.

How to run
- HTML demo
  - Open Pawroutine/pet_care_app_final.html in a browser (double-click or serve it).
- Python CLI
  1. python -m venv .venv && source .venv/bin/activate
  2. python Pawroutine/main.py
  - The CLI uses care_log.csv in the working directory.

Files included
- main.py — the Python CLI (interactive)
- pet_care_app_final.html — UI demo (screenshot referenced in README)
